# 湯切りますたー

## プロジェクト概要
ラーメン店主が主役のカオス系リズムゲーム。
店主（実写ベース）がヌルヌル動き、スコアに応じてカオスなイベントが発生する。

## 重要な方針
- 「笑い」を最優先。真面目にバカをやる
- 店主画像は季記さんがナノバナナで作成して提供
- BGM/SEも季記さんが準備して提供
- 技術的な完璧さより「面白いかどうか」を重視

## 素材の受け取り方
- 画像: `public/images/` 以下に配置
- 音声: `public/audio/` 以下に配置
- 譜面: `public/beatmaps/` 以下にJSON形式で配置

## 技術スタック
- React + TypeScript + Vite
- Tone.js（オーディオ同期）
- Framer Motion（アニメーション）
- Tailwind CSS

## コーディング方針
- コンポーネントは小さく分割
- 日本語コメント推奨
- 面白さに直結する機能を優先

## ディレクトリ構成
```
public/
├── images/
│   ├── chef/           # 店主画像
│   ├── chaos/          # カオス演出画像
│   └── ramen/          # ラーメン進化画像
├── audio/
│   ├── bgm/            # BGM
│   └── se/             # 効果音
└── beatmaps/           # 譜面データ（JSON）

src/
├── components/
│   ├── Game/           # ゲームコア
│   ├── Chef/           # 店主キャラクター
│   ├── Chaos/          # カオスイベント
│   ├── Ramen/          # ラーメン進化
│   └── UI/             # UI要素
├── hooks/              # カスタムフック
├── types/              # 型定義
└── data/               # 静的データ
```

## ゲームシステム

### 判定ウィンドウ
| 判定 | 許容時間 | スコア |
|------|---------|--------|
| PERFECT | ±30ms | 1000点 |
| GREAT | ±60ms | 800点 |
| GOOD | ±100ms | 500点 |
| MISS | それ以外 | 0点 |

### カオスイベント
スコアに応じて発生確率が上昇（最大50%）
- 1000点～: ウインク、軽いネタ
- 3000点～: 猫の竜巻旋風脚など
- 5000点～: 謎のおじさん乱入
- 8000点～: 宇宙服着用
- 10000点～: 丼の中に入る

### ラーメン進化
コンボ数に応じてラーメンがグレードアップ
0→10→20→30→50→70→100コンボで進化
